# CM Usthad Archive Site

This is a polished first version of a professional archive website for CM Usthad, C. M. Abdulla Moulavi.

## Site Structure

- Home hero with portrait, name, and archive positioning
- Key facts section for quick credibility
- Biography and timeline
- Fields of work: fiqh, astronomy, mathematics, languages
- Books and writings with search and filters
- Case diary with neutral, source-led records
- Articles section for essays and research
- Image gallery with caption modal
- News section
- Testimonials section
- Contribution desk for collecting real materials
- Draft source notes

## Design Direction

The design is intentionally calm and archival: deep green, white, charcoal, brass, and restrained accent colors. The page avoids generic AI-style hero blocks and uses a real public portrait from the MIC website as the main visual.
